# Módulo Ra-02

LoRa Ra-02 (433 MHz)

## Conexiones módulo

| Ra-02 | Arduino UNO |
|:-------------:|:--------------:|
| GND   | GND   |
| 3.3 V | 3.3 V |
| RST   | D9    |
| NSS   | D10   |
| MOSI  | D11   |
| MISO  | D12   |
| SCK   | D13   |
| DIO0  | D2    |
